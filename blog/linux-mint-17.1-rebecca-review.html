Linux Mint 17.1 (Rebecca) Review

"Hello World", I have just installed the Linux Mint 17.1 when my Ubuntu got crashed (My mistake :p). So I thought to gave mint a try. This is not full review, but listing some of the things I liked till now:

  • Consumes less RAM than Ubuntu (380MB vs 680MB) - I dont know if thats the same case for everyone or not, but this is helping me to run more programms simultaneous without freezing (My Ubuntu freezes to much since I have only 2GB RAM)
  • Pre-Installed apps - It has some useful pre-installed apps like Hexchat, USB image writer, TomBoy Notes, Pidgin. Which have saved some of my time
  • More Customization - Can change folder to any colors, Nice panel for adding apps, Title Bar button(max, min, close) positions & order, Effects of closing and opening applications. That's kind of pretty small features.
  • Categorization of apps on Menu - Categorized menu to find relevant app in relevent category.
  • By default copies text when you select texts using left mouse button, Paste it using mouse's middle button. That way you can use copy two time simultaneously (Another way could be using CTRL-C & CTRL-V or using Edit Menu.)
