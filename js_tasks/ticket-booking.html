Multiplex Ticket Booking

    Javascript task - Instructions:
  • Creating visual seat picker : define the names of the categories i.e. gold, silver, bronze
  • As well as the number of rows and seats per row in the specific category.
  • Include a form that asks the user the number of tickets they are booking & booking category
  • Should only allow people to book seats that are adjacent to each other
  • Also prevent people from creating single seat silos unless there is no alternative.
    • For example in a row of 4 seats, a user should not be allowed to book seats 2 and 3 because in that case seat 1 would be alone and seat 4 would be alone. He should be forced to book seats 1-2 or seats 3-4.
