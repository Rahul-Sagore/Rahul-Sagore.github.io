Vertical Tabs with Material design lite
Posted - 4th April 2016
Google's Material design lite library has horizontal tabs but doesn't have vertical tabs and the possible reason for this could be : "Material Design Specification doesn't have specification for vertical tabs". Currently In MDL, horizontal tabs looks like this:

MDL's default tab is horizontal. But depending on the design you would like to show it vertically aligned. You can achieve this with "MDL Grid" and "Some Custom CSS". I have used the design of vertical tabs similar to which I have used in past.
Vertical tabs using custom CSS
Implementation
To make it work:
  • Add ".vertical-mdl-tabs" class to ".mdl-tabs", which is a tab container. (I have added vertical tab design with this custom class)
  • Then divide the ".mdl-tabs" and ".mdl-tabs__panel" using "Material Grid columns".(Check HTML tab of codepen)
  • MDL Grid will divide the tab and tab content in two columns and will make it vertical alligned
  • But the design will be the same as horizontal tab. To make it look like vertical tab use the CSS from CSS Tab of the codepen.
This is just a temporary solution. Check the HTML/CSS tab of above codepen to understand more. Comments and improvements are welcomed :)
